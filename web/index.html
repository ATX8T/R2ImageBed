Worker 连接配置

本站静态部署在 GitHub Pages，所有 R2 操作通过你自己部署的 Cloudflare Worker 完成。配置仅保存在浏览器本地（localStorage），不会上传。 查看部署指南 ↗

💡 R2 凭证在哪？
v3.0 静态版本中，R2 桶与凭证直接由 Worker 通过 r2_buckets 绑定，
所以你 不需要 在前端填入 Access Key / Secret，更安全。
R2 桶名、公开域名等在仓库 worker/wrangler.toml 中配置。
